# Noa's Research Inbox

Brian shares links directly to Noa's Telegram channel. This file tracks the inbox processing workflow.

## How It Works

1. Brian sees something interesting (X post, YouTube video, blog, etc.)
2. Brian shares it to Noa's Telegram via iOS share sheet
3. Noa's daily research cron checks Telegram for new shared links
4. For each link: analyze content, decide if it's a one-time item or ongoing source
5. One-time items: include in that night's research brief
6. Ongoing sources: add to SOURCES.md with proper metadata
7. Acknowledge processing in the next Dawn Patrol or research brief

## Processing Rules

- **X/Twitter profiles** → Add to SOURCES.md "Sources to Evaluate" with context. Check their recent posts for relevance.
- **YouTube channels** → Add RSS feed to SOURCES.md primary or secondary sources
- **Blog posts** → Summarize in research brief. If the blog is consistently good, add to SOURCES.md
- **Tools/products** → Evaluate and include in research brief with relevance to SailorSkills
- **Podcasts** → Add RSS feed, summarize recent episodes

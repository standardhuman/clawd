## Design Elements

### Color Palette
- **Ocean Colors**: Deep blues and teals for the nautical theme
- **Wood Tones**: Warm browns for weathered wood textures
- **Accent Colors**: Each agent has their own distinctive accent color
- **Rope/Brass**: Complementary colors for nautical elements

### Typography
- **Headings**: Playfair Display for elegant, nautical feel
- **Body Text**: Inter for clean, readable content
- **Icons**: Font Awesome for consistent iconography

### Interactive Features
1. **Smooth Scrolling**: Navigation between page sections
2. **Hover Effects**: Agent cards and pipeline cards respond to interaction
3. **Parallax**: Subtle background movement for depth
4. **Responsive Design**: Adapts to all screen sizes
5. **Print Styles**: Optimized for printing

## Accessibility

- **Keyboard Navigation**: All interactive elements are keyboard accessible
- **Screen Reader Support**: Proper ARIA labels and semantic HTML

- **High Contrast Mode**: Special styles for high contrast preferences

## Usage

Simply open `index.html` in any modern web browser. No build process or dependencies required.

## Customization

To update agent information:
1. Edit the HTML structure in `index.html`
2. Update agent details in the appropriate card sections
3. Modify colors in the CSS `:root` variables
4. Update avatar image paths if needed

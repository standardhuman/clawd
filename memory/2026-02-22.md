## "Ask Your Dock Neighbors" ‚Äî Social Recommendation Requests SHIPPED

**Commit:** 275c4de

Brian's idea: instead of just reviews, let boat owners actively ASK their trust network for recommendations ‚Äî emulating the real dock-walk/yacht-club-bar pattern.

**The flow:**
1. Owner creates request: "I need a rigger for standing rigging inspection" + boat context + marina + urgency
2. Visibility scoping: trust_circle (mutual vouches only) / marina (all verified at same marina) / all_verified
3. Auto-fan-out trigger populates recipient list based on owner_vouches + boat_owner_verifications
4. Neighbors see incoming requests, respond with: provider name, personal note, confidence level, phone, "used personally" flag
5. Requester sees curated recommendations from people they actually trust
6. Can mark fulfilled (with which provider) or close

**This is the real competitive moat.** Reviews are Yelp. This is the actual behavior pattern that makes marina communities work. No one else has this.

**DB:** 3 new tables + 2 triggers (fan-out + mark-responded) + full RLS
**UI:** /ask page with create/incoming/my-requests sections, homepage feature card

- **Idea:** Study of 6,000 CEOs finds AI has had no impact on employment or productivity. ### CEL (Common Expression Language) for Business Logic
- **Source:** https://celbyexample.com/
- **Summary:** CEL is a portable expression language used by Kubernetes (admission control), Google Cloud IAM, Firebase security rules, and Envoy Proxy. It evaluates expressions against data (JSON, Protobuf) ‚Äî fast, safe, sandboxed.
- **Action:** Replace custom validation logic in SailorSkills with CEL expressions. Example: "client.subscription_status == 'active' && client.balance >= 0". Could be used for pricing rules, access control, scheduling constraints. Self-contained, auditable, testable.

### Cognitive Debt vs. Technical Debt
- **Source:** Simon Willison blog (multiple entries on cognitive debt)
- **What it does:** New concept: when AI makes coding fast but humans lose understanding of the system. "Cognitive debt" accumulates when teams can't explain why design decisions were made or how system parts work together.
- **How we'd use it:** Framework for evaluating agent-built code. Need to ensure Howard and other agents maintain system understanding, not just produce working code. Could implement "cognitive debt audits" in our development process.

### Qwen3.5: 397B MoE Model with 17B Active Parameters
- **Source:** Simon Willison blog
- **What it does:** Alibaba's Qwen3.5-397B-A17B is a Mixture of Experts model where only 17B parameters activate per forward pass. Open weights, multi-modal (vision), 807GB on Hugging Face.
- **How we'd use it:** For cost-sensitive inference tasks where we need vision capabilities. The MoE architecture means lower inference cost despite large model size.

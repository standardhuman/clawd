## ⚓ SailorSkills Augmentation

### Weave — Language-Aware Merge for Multi-Agent Development
- **Source:** https://github.com/Ataraxy-Labs/weave (106 pts, 62 comments)
- **Summary:** Weave is a git merge driver that understands code structure at the entity level (functions, classes, methods) instead of lines. When two agents add different functions to the same file, git creates a conflict; Weave merges them cleanly because it knows they're separate entities. It ships as an MCP server with 14 tools, enabling agents to claim entities before editing and check for conflicts preemptively. Endorsed by Elijah Newren (git merge-ort creator) and the GitButler team.
- **Action:** Install Weave on Brian's development environment for any multi-agent SailorSkills development work. When using OpenClaw sub-agents to work on the Pro app or marketplace in parallel (e.g., one agent on the booking flow, another on the service logging), Weave prevents the merge conflicts that currently require manual resolution. The MCP server integration means agents can coordinate automatically.
